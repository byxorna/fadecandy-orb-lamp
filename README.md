# Config

* LAYOUT - layout file (./layout_16x8z.json)
* OPC_HOST - OPC server host (localhost)
* OPC_PORT - OPC server UDP port (7890)
* PORT - http listen port
* PATTERNS_DIRECTORY - directory to pull in pattern functions (./patterns)
